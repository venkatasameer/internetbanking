Internet Banking Project - Setup Guide

Step 1: Install XAMPP

To run the Internet Banking project locally, you need to install XAMPP, which includes Apache, MySQL, and PHP. You can download XAMPP from the official website: XAMPP Download.

Follow the installation instructions on the website. Once installed, open the XAMPP Control Panel and start the Apache and MySQL services.

Step 2: Setting Up Your Project in XAMPP

After installing XAMPP, navigate to the htdocs folder within the XAMPP installation directory (e.g., C:\xampp\htdocs). This is where you should place your PHP project files.

You can either clone your GitHub repository directly into the htdocs folder, or you can manually download and place your project files there.

To clone your GitHub repository, open the command prompt, navigate to the htdocs directory, and run: 

git clone https://github.com/venkatasameer/internetbanking.git

Step 3: Accessing the Project Locally

Once your project files are in the htdocs folder, open your web browser and go to http://localhost/internetbanking/index.php to access your project.

Alternatively, you can view the index.php file directly on GitHub by clicking the link below:

View index.php on GitHub

Step 4: Working with MySQL

XAMPP includes phpMyAdmin, a web-based interface to manage MySQL databases. You can access phpMyAdmin by navigating to http://localhost/phpmyadmin in your browser.

Use phpMyAdmin to create a new database for your Internet Banking project. You can then connect your PHP project to this database using the following code: 

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "your_database_name";

// Create connection
$conn = new mysqli($servername, $username, $password, $dbname);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
echo "Connected successfully";
?>

Replace your_database_name with the name of the database you created in phpMyAdmin.
