Welcome to the PHP, XAMPP, and SQL Guide

This guide will help you set up and run PHP on your local machine using XAMPP. Additionally, you'll learn how to connect PHP to a MySQL database.

Step 1: Install XAMPP

Download and install XAMPP from the official website: XAMPP Download. XAMPP includes Apache (a web server), MySQL (a database), and PHP (a scripting language), making it a convenient all-in-one package for local development.

Step 2: Setting Up Your PHP Project

After installing XAMPP, navigate to the htdocs folder within the XAMPP installation directory (e.g., C:\xampp\htdocs). This is where you should place your PHP files.

Create a new folder for your project (e.g., myproject), and within that folder, create an index.php file.

Step 3: Running Your PHP Script

To run your PHP script, start the Apache server from the XAMPP Control Panel. Then, open your web browser and go to http://localhost/myproject/index.php.

Step 4: Working with MySQL

XAMPP also includes phpMyAdmin, a web-based interface to manage MySQL databases. Access phpMyAdmin by navigating to http://localhost/phpmyadmin.

Create a new database and use the following code to connect PHP to MySQL: 

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "your_database_name";

// Create connection
$conn = new mysqli($servername, $username, $password, $dbname);
